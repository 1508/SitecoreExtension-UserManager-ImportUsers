SitecoreExtension UserManager ImportUsers
=========================================

Allow developers to import users and their roles to sitecore from a csv file. 

Created for simplying user management and roles bindings, can be run multiple times to reset roles. 
Has the option the send out a introductionary e-mail to the users.

Interface is very raw and is only meant to be used by developers.
